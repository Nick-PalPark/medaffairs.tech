How to Use This Tool

This tool generates GitHub API calls to add articles to your website. After filling out the form:

  1. Click "Generate API Call" to create the repository dispatch payload
  2. Copy the generated curl command
  3. Run it in your terminal or use a tool like Postman
  4. The article will be processed and added to your website automatically
Note: You'll need a GitHub Personal Access Token with repo permissions to use the generated API calls.
